# 🏏 Cricbuzz Clone

A web-based application that provides live cricket scores, match updates, team details, and player statistics — inspired by Cricbuzz.

---

## 🚀 Features

* 📊 Live match score updates
* 🗓️ Match schedules and results
* 👥 Team and player information
* 📈 Points table and rankings
* 🔍 Search functionality for matches/players
* 📱 Responsive UI for mobile and desktop

---


## 📌 Future Enhancements

* Real-time notifications
* User login & favorites
* Commentary section
* Fantasy league integration
